# ADR-0001: The directive to 'construct an answer from the profile / extrapolate ...

- Status: proposed
- Date: 2026-06-30

## Context
Raised during spec: The directive to 'construct an answer from the profile / extrapolate rather than guess or defer' risks colliding with the absolute carve-out escalation path (money, credentials, permissions, external publishing, deletion). 'Never defer' must not be read as 'never escalate' — carve-outs must still be escalated to the real user, never answered by extrapolation.

## Decision
Scope the extrapolation directive explicitly to novel technical/decision questions, and keep the absolute-carve-out escalation rule as an overriding exception so 'don't defer' never overrides 'escalate carve-outs'.
